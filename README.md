# ✍️ Sentiric Knowledge Indexing Service

RAG mimarisinin 'Yazma' (Indexing) motorunu temsil eder. Verileri Qdrant üzerinde indeksler.

## 🚀 Hızlı Başlangıç
```bash
python -m app.runner
```

## 🏛️ Mimari ve Mantık
* **Geliştirici Kuralları:** Gizli [.context.md](.context.md) dosyasını okuyun (AI Ajanları için zorunludur).
* **İş Mantığı ve Algoritmalar:** [LOGIC.md](LOGIC.md) dosyasını inceleyin.
* **Anayasal Konum:** [sentiric-spec/spec/services/knowledge-indexing.spec.yaml](https://github.com/sentiric/sentiric-spec)
