# 🧠 AI SYSTEM INSTRUCTIONS FOR KNOWLEDGE-INDEXING-SERVICE
1. **Teknoloji:** Python 3.11+, FastAPI, gRPC, Qdrant, SentenceTransformers, Asyncpg.
2. **Read Yasağı (CQRS):** RAG araması (Query) işlemi bu serviste KESİNLİKLE YASAKTIR. Bu servis sadece veriyi çeker, parçalar (chunking), vektörleştirir ve Qdrant'a YAZAR.
3. **Durum (State) Yönetimi:** Gerçek zamanlı çağrı oturumu (CallContext) veya kullanıcı durumu (Session) yönetemez.
4. **Loglama:** `print()` yasaktır. `structlog` kullanılarak SUTS v4.0 JSON formatı basılmalı ve Infinite Loop engellemek için `WriteLoggerFactory` kullanılmalıdır.
